# 🌦️ Atmosphere — Modern Glassmorphic Weather App

A sleek, responsive, modern Weather Application built with **Vanilla JavaScript**, **CSS3 (Glassmorphism & Custom Properties)**, and **HTML5**. It fetches real-time meteorological data using the **WeatherAPI** service.

## ✨ Features

- 💎 **Modern Glassmorphism UI**: High-end translucent cards, backdrop blur, ambient background glow, and smooth micro-animations.
- ⚡ **Real-Time Weather Metrics**: Live temperature, weather condition text, city & country name, humidity, wind speed, "feels like" temperature, and air pressure.
- 📱 **100% Mobile & Desktop Responsive**: Fluid layout designed to work seamlessly on screens of all sizes (mobile, tablet, desktop).
- ⌨️ **Keyboard Accessible**: Search triggering on both button click and `Enter` key submit.
- 🛡️ **Robust Error Handling**: Friendly error toasts for invalid city searches or network disconnections without app crashes.
- ⏳ **Loading State Indicator**: Smooth CSS spinner state while waiting for network requests.
- 🚀 **Zero Dependencies**: Built with 100% standard web technologies (No heavy frameworks or libraries required).

---

## 🛠️ Tech Stack & Architecture

- **Structure**: Semantic HTML5 markup (`<main>`, `<form>`, `<section>`).
- **Styling**: Vanilla CSS3 using custom properties (variables), Flexbox, CSS Grid, animations, and `@media` queries.
- **Typography**: Google Font (*Outfit*).
- **Scripting**: Asynchronous JavaScript (`async/await`, `fetch API`, DOM manipulation, and defensive error handling).
